npm-check是用来检查npm依赖包是否有更新，错误以及不在使用的，我们也可以使用npm-check进行包的更新。

## 安装npm-check

```javascript
npm install -g npm-check
```

## 检查 npm 包的状态

```javascript
npm-check -u -g
```

通过上下键可以移动光标，使用空格键可以选择需要处理的包，回车直接进行处理。
